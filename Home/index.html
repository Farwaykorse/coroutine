    Library Features

    I've used the C++ coroutine since 2017.02.
    Currently working to follow the C++ 20 and writing examples in my style.

    Coroutine Handle

    Works related to <coroutine> header are placed in 1 file.

    #include <coroutine/frame.h>
using namespace std::experimental;
    • coroutine_handle<P>
      • Frame prefix
    • coroutine_traits<R, P...>
    • suspend_never and suspend_always

    Network I/O

    The socket async I/O related features are adopting Guideline Support Library.
    The type/functions don't have namespace for now. (They are global !)

    #include <coroutine/net.h>
using namespace gsl;
